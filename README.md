# H.E.R.M.E.S


Description:
H.E.R.M.E.S. is a mobile environmental monitoring embedded system designed for residential spaces. The system collects air quality, CO₂, CO, and temperature data across multiple areas of a residential area using onboard sensors and displays the data in various ways. Environmental readings are processed locally on embedded hardware to evaluate safety, calculate cumulative exposure over time within a residential space, and classify the readings into simple risk levels for better understanding. The system autonomously navigates between locations, detects persistent unsafe conditions, and provides both local and remote alerts. The architecture prioritizes reliability, autonomy, and minimal user interaction while providing meaningful environmental health insights to users.
